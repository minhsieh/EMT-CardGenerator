EMScout Card Generator
========================

這是一個在台北市91團營隊上撰寫的一個網頁
可以用來產生那次營隊的名牌
並且可以讓使用者自行上傳圖片並填入自己的資料

這次主要練習的技術是圖像處理以及裁切的技巧
